
# Intelligent Candidate Discovery

## 📌 Project Overview
Intelligent Candidate Discovery is an AI-powered application that helps recruiters identify the most suitable candidates by comparing resumes with a job description using Natural Language Processing (NLP) and Machine Learning.

## 🚀 Features
- Upload multiple resumes
- Enter a job description
- Match resumes with the job description
- Rank candidates based on similarity
- Simple and user-friendly Streamlit interface

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- NumPy
- PyPDF2
- Sentence Transformers
